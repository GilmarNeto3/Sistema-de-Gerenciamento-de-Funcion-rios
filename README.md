# Sistema de Gerenciamento de Funcionários

## Descrição

Projeto em TypeScript para gerenciar funcionários de uma empresa.
Permite cadastrar, atualizar salário e consultar informações.

---

## Funcionalidades

* Cadastrar funcionários
* Atualizar salário
* Consultar funcionário

---

## Como executar

1. Compilar:

```
tsc
```

2. Executar:

```
node dist/main.js
```
